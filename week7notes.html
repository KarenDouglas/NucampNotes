Week 7 Notes: React/Redux

Intro to React & Front-End JS Libraries/Frameworks

Framework vs Library

  • Inversion of Control: with Libraries you have contol, you call the library when you want to use it. On the other hand, Frameworks has control so it calls your code when needed aka "The HollyWood Principle"(don't call us, we'll call you)

A Brief History of React

  • First developed by Jordan Walke at Facebook
  • Released as open source in 2013
  • Designed for speed, simplicity, scalability/

Defining Features of React

  • Uses a declarative approach, so it acts as a mediator between you and the DOM
  • Uses a Component-Based approach
  • It makes no assuptions about the rest of you tech stack-focuses on the UI and lets you decide the rest of the app architecture

declarative approach

you declare what you want to see, the libraries like React handles updating the DOM in a consisten way.

Imperative approach

you manipulate the DOM directly to create, insert , remove nodes and spell out every step yourself

Component Based Approach

encapsulates the UI into units call Components

Intro to JSX

JSX

JSX

a syntactic extension to JavaScript created to help write front-end code faster

  • using JSX you can write, what looks like html, inside of JS
  • its like syntactic sugar, or a preprocessor language for JS similar to Sass for CSS
  • you can write your code in one way, then it gets compiled to regular JS on its way to the browser
  • many HTML attributes are the same
  • Some attributes are different because they already have meaning in JS
    • You can embed JS expression inside JSX using curly braces {}
  • When an element has no content, you can write it as a self-closing tag in JSX, even though that is not possible in HTMl

JS Import and Export

Import & Export

  • import and export keywords in JS all files to share data with each other
  • the share ddata can be objects, primitives and function
  • there are two kins of exports: naed and default
    • named can have many named exports, while default can only have one default export
